# MiniAlgo Compiler

## Description
Ce projet est un mini-compilateur pour le langage **MiniAlgo**, développé avec **Flex** et **Bison**.  
Il couvre toutes les phases de compilation : analyse lexicale, syntaxique et sémantique, gestion de la table des symboles, génération de code intermédiaire, et détection des erreurs.

---

## Objectifs du projet
1. **Analyse lexicale** : reconnaissance des mots-clés, identificateurs, constantes, opérateurs et délimiteurs.
2. **Analyse syntaxique et sémantique** : vérification de la structure du programme et des instructions.
3. **Table des symboles** : stockage des variables, constantes et tableaux avec recherche et insertion.
4. **Génération de code intermédiaire** : création de quadruplets.
5. **Gestion des erreurs** : détection des erreurs lexicales, syntaxiques et sémantiques.
